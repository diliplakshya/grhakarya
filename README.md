<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com:siddhatva/grhakarya.git
```

Go to the project directory

```bash
  cd grhakarya
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
